# Semicolon; Team Page

세미콜론 팀 페이지는 Git 협업 실습과 웹페이지 제작을 연습하기 위해 만든 정적 웹사이트입니다.

팀원들이 각자 브랜치에서 개인 프로필 페이지를 만들고, 메인 페이지에 멤버 소개와 2026년 1학기 활동 성과를 연결하면서 Git/GitHub 협업 흐름을 익히는 것을 목표로 합니다.

## 웹 접속 안내

이 프로젝트는 README보다 실제 웹 화면에서 보는 것을 권장합니다. 메인 화면에서는 동아리 소개, 팀원 카드, 1학기 활동 성과를 볼 수 있고, 각 팀원 카드에서 개인 프로필 페이지로 이동할 수 있습니다.

| 구분 | 링크 |
| :--- | :--- |
| 라이브 사이트 | Vercel 배포 후 링크 추가 예정 |
| 로컬 메인 페이지 | `index.html` |
| 팀원 프로필 페이지 | `pages/` 폴더 |

Vercel 배포 후 아래 칸에 실제 주소를 넣으면 됩니다.

```md
| 라이브 사이트 | https://your-vercel-domain.vercel.app/ |
```

## 이 웹사이트는 무엇인가요?

이 웹사이트는 세미콜론 동아리의 팀 소개 페이지입니다.

- 메인 페이지에서 세미콜론 동아리 소개를 확인할 수 있습니다.
- 멤버 카드에서 각 팀원의 개인 프로필 페이지로 이동할 수 있습니다.
- 2026년 1학기 활동 성과 섹션에서 정기 미팅, 정기 스터디, 공모전 참가, Git 협업, LLM 챗봇 구현 등 주요 활동을 확인할 수 있습니다.
- 일부 개인 페이지에서는 팀원이 진행한 작업물과 외부 링크를 확인할 수 있습니다.
- 모바일 화면에서도 볼 수 있도록 반응형 스타일을 적용했습니다.

## 디렉토리 구조

```text
SEMICOLON-TEAMPAGE
├─ css/              # 스타일시트 파일
│  ├─ index.css      # 메인 페이지 스타일
│  └─ [name].css     # 팀원 개별 프로필 전용 스타일
├─ img/              # 이미지 에셋
├─ pages/            # 팀원 개별 프로필 HTML 페이지
│  └─ [name].html    # 팀원 개별 프로필 파일
├─ index.html        # 팀 소개 메인 페이지
└─ README.md         # 프로젝트 소개 및 협업 컨벤션
```

## 협업 컨벤션

### 파일 관리

- HTML은 페이지 역할에 맞게 `index.html` 또는 `pages/` 폴더에 작성합니다.
- CSS는 `css/` 폴더의 외부 스타일시트로 분리합니다.
- 이미지가 필요한 경우 `img/` 폴더에 저장하고 상대 경로로 연결합니다.

### Git 작업 방식

| 브랜치명 | 용도 |
| :--- | :--- |
| `main` | 최종 배포 및 병합용 |
| `feature/이름` | 기능 추가 및 개인 작업 |

커밋 메시지는 `타입: 제목` 형식으로 작성합니다.

| 타입 | 설명 |
| :--- | :--- |
| `feat` | 새로운 기능 추가 |
| `fix` | 버그 수정 |
| `design` | 디자인 변경 |
| `docs` | 문서 수정 |
| `refactor` | 코드 리팩토링 |
| `chore` | 단순 작업 |
